Home Page

The purpose of this project was to analyse how weather changes as you get closer to the equator. To accomplish this analysis , I first pulled data from the OpenWeatherMap API to assemble a dataset on 604 cities.

After assembling the dataset, I used Matplotlib to plot various aspects of the weather vs the latitude. Factors I focused on were maximum temperature, humidity, cloudiness and wind speed. This site provides the source data and visualistions created as part of the analysis, as well as explanations and descriptions of any trends and correlations I witnessed.
